# Portifólio

Projeto pessoal de portfólio desenvolvido para demonstrar projetos, layout responsivo e uso de recursos estáticos (CSS, JavaScript e fontes).

## Descrição

Este repositório contém a versão estática de um portfólio construído com HTML, CSS e JavaScript puros. A página principal é index.html e os estilos e scripts estão organizados na pasta `src`.

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome (local)

## Estrutura do projeto

- [index.html](index.html) — arquivo principal da página.
- [src/css/style.css](src/css/style.css) — estilos principais.
- [src/css/menu.css](src/css/menu.css) — estilos do menu.
- [src/css/reset.css](src/css/reset.css) — reset de estilos.
- [src/css/variable.css](src/css/variable.css) — variáveis CSS.
- [src/js/index.js](src/js/index.js) — script principal.
- [fontawesome/](fontawesome/) — assets do Font Awesome (CSS, JS e fontes).

## Como executar

1. Abra o arquivo [index.html](index.html) no navegador (duplo clique ou via "Abrir com").
2. Para desenvolvimento local com live-reload (opcional), use uma extensão/servidor local como Live Server no VS Code ou execute um servidor HTTP simples:

```bash
# com Python 3
python -m http.server 3000

# então abra http://localhost:3000
```

## Personalização

- Substitua o conteúdo em index.html e os arquivos em `src/` para atualizar textos, imagens e estilos.
- O diretório `images/` contém as imagens utilizadas; atualize conforme necessário.
